## Project Title

# 🚀 Linux DevOps Administration & Automation on AWS

## Overview

The Linux DevOps Administration & Automation on AWS project is a hands-on implementation of Linux system administration, server troubleshooting, Bash scripting, Git workflow management, and Cron job automation using an AWS EC2 Ubuntu Server.

The primary objective of this project is to simulate real-world Linux administration tasks performed by System Administrators and DevOps Engineers. Instead of focusing only on theoretical concepts, the project emphasizes practical implementation, automation, troubleshooting, and documentation.

Throughout the project, a secure Linux environment was configured by creating users and groups, managing file permissions, implementing shared directories, monitoring server health, automating repetitive tasks with Bash scripts, scheduling jobs using Cron, and managing source code with Git and GitHub.

The project follows production-oriented best practices, including the Principle of Least Privilege (PoLP), structured logging, automated backups, system monitoring, and organized documentation.

This repository serves as both a learning resource and a portfolio project demonstrating practical Linux administration and DevOps skills.
